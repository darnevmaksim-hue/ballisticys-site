Что делает мод и зачем он нужен

Ballisticys моды добавляют клиентские инструменты для точной стрельбы и контроля целей: баллистический калькулятор, мини-HUD, экран истории, PVO-интерфейс, захват/сброс цели и команды управления. Это снижает ошибки в расчётах и ускоряет прицеливание в сценариях с артиллерией и автопушками.

Быстрый гайд

Важно: файл мода .jar может открываться архиватором как zip внутри, это нормально. Его не нужно распаковывать.

Fabric установка

  1. Установи Fabric Loader для 1.20.1.
  2. Положи jar в папку mods.
  3. Убедись, что установлен Fabric API.

Forge установка

  1. Установи Forge 47+ для 1.20.1.
  2. Положи Forge jar в папку mods.
  3. Запусти клиент на Java 17.

Injector запуск

  1. Скачай и распакуй toolkit zip.
  2. Запусти Start-Ballisticys.ps1.
  3. Проверь inject-launch.log после запуска.
